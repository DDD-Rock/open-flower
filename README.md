# 枫灵 MapleKeeper

> 某横版网游祭司职业自动Buff释放助手

## 项目简介

**枫灵 MapleKeeper** 是一个用于某横版网游中祭司职业的自动Buff释放辅助工具。通过图形化界面配置Buff按键和持续时间，程序会自动计时并循环释放技能。支持两种运行模式：

- **死花模式**：Buff到期后自动离开市场→移动到刷怪点→释放Buff→返回市场等待
- **活花模式**：在当前位置自动循环释放Buff

## 功能特性

### 核心功能
- ✅ 支持最多 **6 个Buff** 独立配置（按键 + 持续时间）
- ✅ **死花 / 活花** 双模式运行
- ✅ 自动倒计时，Buff到期精准释放
- ✅ 游戏窗口自动识别（模糊匹配窗口标题）
- ✅ 设置持久化（INI 配置文件），重启不丢失

### 死花模式专属
- ✅ **小地图识别**：通过模板匹配检测市场Logo，判断当前场景
- ✅ **自由市场按钮检测**：多尺度模板匹配，自动定位并点击
- ✅ **传送门导航**：小地图角色点追踪，自动走向传送门进入/离开市场
- ✅ **手动标记传送门**：支持在小地图上手动标注传送门位置
- ✅ **对话框检测**：自动识别并确认传送门对话框
- ✅ **出市场后移动模式**：右→左 / 仅左 / 仅左(魚窩) 多种可选

### 拟人化机制
- ✅ 随机延迟秒数（可配置比例）
- ✅ 释放技能前/后随机左右走动
- ✅ 空闲时自动坐椅子（可配置椅子按键）
- ✅ 移动前跳跃拟人化处理
- ✅ 多次点击模拟（避免单次点击不生效）

### UI 界面
- ✅ PyQt6 图形化界面，操作直观
- ✅ 虚拟键盘选择按键，无需手动输入
- ✅ 实时日志输出，运行状态一目了然
- ✅ 倒计时显示，可视化等待进度
- ✅ 一键启动 / 停止

## 安装说明

### 环境要求

- **Python** 3.10+
- **操作系统**：Windows（依赖 `pywin32` 和 `keyboard` 库）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

> ⚠️ 程序需要 **管理员权限** 才能模拟键盘输入，可使用项目中的 `start_admin.bat` 以管理员身份启动。

## 使用指南

### 1. 识别游戏窗口

程序启动时会自动尝试识别游戏窗口。如未找到，可点击 **"识别窗口"** 按钮手动识别。

### 2. 配置Buff

在设置面板中为每个Buff槽位配置：
- **启用/禁用** 复选框
- **按键** — 点击按钮弹出虚拟键盘选择
- **持续时间（秒）** — Buff的持续时间

### 3. 选择运行模式

- **勾选"释放后回到市场"** → 死花模式
- **不勾选** → 活花模式

### 4. 死花模式额外配置

- **攻击键 / 跳跃键**：用于出市场时的移动操作
- **出市场后移动方式**：右→左 / 仅向左 / 仅向左(魚窩)
- **坐椅子**：空闲等待时自动坐下，需配置椅子按键
- **标记传送门**：可在小地图上手动标注传送门位置，提升导航精度

### 5. 启动运行

点击 **"开始运行"**，程序进入自动循环：
- 活花：到期 → 释放Buff → 等待 → 循环
- 死花：等待 → 离开市场 → 移动 → 释放Buff → 返回市场 → 循环

## 打包为 EXE

### 安装 PyInstaller

```bash
pip install pyinstaller
```

### 打包命令


打包完成后，程序文件夹位于 `dist/苗圃助手/`，运行 `苗圃助手.exe` 即可。

### Visual C++ 运行库

如果目标机器提示缺少运行库，请下载安装 [VC++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)。
